Designers Debate What's Left When AI Writes the Code in a Day

The floor for user interface quality has never been higher—and that's forcing designers to confront an uncomfortable question.

Date: February 10, 2026
Type: Socratic Discussion
Co-Hosts: Sahil Chaturvedi, Austin Kelsay

That tension surfaced throughout a February meetup at Bitcoin Park Austin, where product designers and developers grappled with tools that are rapidly rewriting their workflows. "I cannot believe that just one year ago, we used to spend hours, weeks, and months making prototypes in Figma," one designer noted in a discussion thread. The group examined whether traditional visual design tools still matter when code generation has become so cheap and fast.

The debate isn't purely theoretical. Benji Taylor, head of design at Base and creator of the acclaimed Family wallet, recently built an intricate icon morphing system in a single day using Claude Code. His achievement—20 icons that transform into any other through actual shape manipulation rather than opacity fades—would have taken substantially longer using traditional methods. Taylor's insight wasn't just prompting the AI to "make animations look nice." He constrained the system to exactly three SVG lines per icon, forcing elegant solutions the AI might not have discovered alone.

"That's the human ingenuity. Claude did all of the hard work, but there is this genius idea: constrain it to three lines. An AI would never do that. Not yet."

Yet the group struggled with where this leaves visual design tools like Figma. One designer argued that creative tools remain essential for work "at the edge of what is possible to imagine and not limited by code." Tools like Unicorn Studio, which enable shader programming and experimental interfaces, represent the "wild cutting edge" that purely code-based workflows might miss. But another attendee countered that such specialized work represents "0.1% of what anyone actually does."

The emerging middle ground involves hybrid tools that preserve the infinite canvas designers love while integrating AI generation. Pencil, an infinite canvas for Claude Code, attempts this balance. "Give me the infinite canvas, give me the ability to iterate on multiple ideas that might be across different branches, and then give me the ability to reach that place," one designer explained. The appeal is "vibe coding" the design while maintaining a visual reference that serves as the source of truth.

Austin Kelsay, an engineer at Voltage presenting at the meetup, pushed the concept further with Sanctum, an admin configuration tool he built for activists. Rather than forcing non-technical users through traditional UI panels, Kelsay envisions administrators simply chatting with AI to configure entire systems. "My dream for what a non-technical admin would be able to do," he explained, is typing requests like "please set up all the styling and basic configs" and letting AI handle implementation. One attendee questioned whether the traditional UI should exist at all: "Why even expose the UI? Why not just make it all the chat?"

The precision problem remains unsolved. Taylor developed Agentation specifically because vague prompts like "the button hover feels off" waste time. His tool lets designers pause animations mid-transition, click specific elements, and generate detailed feedback including class names, file positions, and timing data. "There used to be a lot of dev tools and design tools that would let you do this manually," one attendee noted. "You're still talking to the model, but with precision versus just like, make it better."

What designers seem to be mourning isn't Figma itself but its irrelevance to actual output. "Figma is just now a prompt," one designer argued. "It's not a source of truth." The challenge is creating workflows where design decisions—whether made visually or conversationally—automatically update production code. Several attendees described their current approach: generating multiple AI variations, creating temporary comparison pages, and iterating rapidly through options that would have taken days to prototype manually.

Whether conversational interfaces fully replace visual design tools depends partly on what designers are actually doing. For standard CRUD applications and financial interfaces, AI-generated code may suffice with minimal human guidance. For experimental work pushing technical boundaries, visual creativity tools likely retain purpose. The real question may be how long that distinction lasts.

"We're mourning the death of Figma because it has no relevance to the actual output."

The group left uncertain whether that death is final or if new hybrid tools will resurrect the visual canvas with AI as collaborator rather than replacement.
